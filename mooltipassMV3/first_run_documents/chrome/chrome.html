Thanks for downloading the Mooltipass Extension

It is strongly recommended to disable all other password management applications to prevent potential incompatibilities with the Mooltipass ecosystem.

Part 1: Disabling 3rd Party Password Managers

Step 1

Click the menu icon ( ) on the top right corner of your browser. On the drop down menu that appears, choose "More tools", then "Extensions". This brings you to Google Chrome's extension management page.


Step 2

Find other password managing applications, remove them by clicking the dedicated button:

Be sure to repeat these steps for all other password extensions.

Part 2: Disabling Chrome's Password Manager

You might have noticed that Google Chrome browser asks you if you would like to store credentials, even if you have not installed any password management extensions.

Step 1

Click the menu icon ( ) on the top right corner of your browser. On the drop down menu that appears, choose "Settings".


Step 2

In the settings page of Google Chrome, click the passwords button. It looks like this:

Step 3

Uncheck both sliders for "Offer to save passwords" and "Auto Sign-in":
